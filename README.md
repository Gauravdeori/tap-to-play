# 🎮 TapPlay - Mini Games Hub

A vibrant, modern collection of addictive mini-games built with React and Tailwind CSS. TapPlay offers a variety of quick, fun challenges ranging from strategy and memory to reaction and speed.

## ✨ Features

- **13+ Unique Games**: A diverse collection of mini-games.
- **Modern UI/UX**: Sleek, responsive design with dark/light mode support.
- **Leaderboard**: Track your best scores across different games.
- **Smooth Animations**: Powered by Framer Motion for a premium feel.
- **Responsive Design**: Play on desktop or mobile devices.

## 🕹️ Games Included

- **❌ Tic Tac Toe**: 2 player or vs AI. Classic strategy!
- **✊ Rock Paper Scissors**: Beat the computer. Best of rounds!
- **🃏 Memory Flip**: Match pairs. Test your memory!
- **🔢 Number Guessing**: Guess the secret number!
- **⚡ Reaction Time**: How fast can you react?
- **🐍 Snake**: Eat, grow, survive!
- **🎯 Aim Trainer**: Click targets as fast as you can!
- **👆 Tap Speed**: Tap as fast as possible!
- **🧱 Brick Breaker**: Smash all the bricks!
- **🐹 Whack-a-Mole**: Whack the moles before they hide!
- **🎨 Color Match**: Does the color match the word?
- **⌨️ Typing Speed**: Type words as fast as you can!
- **🧮 Math Sprint**: Solve math problems fast!

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, Lucide React (Icons)
- **State & Data**: TanStack Query (React Query)
- **Animations**: Framer Motion
- **Components**: Shadcn/UI (Radix UI)
- **Testing**: Vitest, Playwright

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [bun](https://bun.sh/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Gauravdeori/tap-to-play.git
   cd tap-to-play
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:8080`

## 📜 Available Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run lint`: Runs ESLint for code style checking.
- `npm run test`: Runs unit tests using Vitest.
- `npm run preview`: Previews the production build locally.
